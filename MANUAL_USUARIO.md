# Manual do Usuário - Agendador

## Introdução

O Agendador é um sistema de agendamento logístico que permite gerenciar agendamentos de transporte para diferentes empresas. Este manual orienta os usuários sobre como utilizar o sistema, dividido por perfis de acesso: Visualizador, Colaborador (Colab) e Administrador (Admin).

### Acesso ao Sistema

1. Acesse a aplicação no navegador: `https://agenda.logtudo.com.br/` (ou o URL fornecido).
2. Faça login com suas credenciais (username e senha).

## Perfil: Visualizador

O Visualizador tem acesso apenas para visualizar dados e relatórios. Não pode criar ou editar agendamentos.

### Funcionalidades Disponíveis

#### 1. Dashboard
- Visualize métricas gerais, gráficos de evolução diária e distribuição por empresa.
- Aplique filtros por empresa, data, perfil e UF.

Passos:
1. Use os filtros no topo para refinar os dados.
2. Observe os gráficos e métricas exibidas.

![Dashboard Visualizador](image/dashboard.png)

*Imagem 2: Dashboard para Visualizador.*

![Dashboard Visualizador](image/dashboard_metrics.png)

*Imagem 2.1: Dashboard para Visualizador.*

#### 2. Histórico de Agendamentos
- Visualize a lista de agendamentos passados.
- Aplique filtros por empresa e data.

Passos:
1. Clique em "Histórico" no menu lateral.
2. Use os filtros para buscar agendamentos específicos.
3. Visualize os detalhes de cada agendamento.

## Perfil: Colaborador (Colab)

O Colaborador pode visualizar dados e criar novos agendamentos.

### Funcionalidades Disponíveis

#### 1. Dashboard (igual ao Visualizador)
- Mesmo acesso de visualização.

#### 2. Histórico de Agendamentos (igual ao Visualizador)
- Mesmo acesso de visualização.

#### 3. Criar Novo Agendamento
- Crie agendamentos para empresas específicas.

Passos:
1. Clique em "Novo Agendamento" no menu lateral.
2. Selecione a empresa, UF e data.
3. Preencha os status com quantidades e perfis.
4. Clique em "Salvar Agendamento".

![Novo Agendamento](image/novo_agendamento.png)

*Imagem 4: Formulário de novo agendamento.*

## Perfil: Administrador (Admin)

O Administrador tem acesso completo: visualizar, criar, editar e gerenciar configurações do sistema.

### Funcionalidades Disponíveis

#### 1. Dashboard (com edição)
- Visualize métricas.
- Edite agendamentos existentes (botão "Editar" nos cards).

Passos para editar:
1. No Dashboard, clique em "Editar" em um agendamento.
2. Modifique os dados no modal.
3. Salve as alterações.

![Editar no Dashboard](image/edit_dashboard.png)

#### 2. Histórico (com edição)
- Visualize e edite agendamentos.

Passos:
1. No Histórico, clique em "Editar" em um agendamento.
2. Modifique no modal.
